知识测验

有氧呼吸的三个阶段按顺序是？
糖酵解 → 丙酮酸氧化 → 电子传递链
丙酮酸氧化 → 糖酵解 → 电子传递链
电子传递链 → 糖酵解 → 丙酮酸氧化
糖酵解 → 电子传递链 → 丙酮酸氧化
正确！

有氧呼吸的正确顺序是糖酵解（细胞质）、丙酮酸氧化（线粒体基质）、电子传递链（线粒体内膜）。
