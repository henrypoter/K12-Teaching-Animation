当前模式说明

原电池模式：化学能转化为电能的自发装置。锌片（负极）失去电子发生氧化反应：Zn → Zn²⁺ + 2e⁻；电子通过导线流向铜片（正极），溶液中H⁺在铜片表面得电子发生还原反应：2H⁺ + 2e⁻ → H₂↑。阳离子向正极移动，阴离子向负极移动，形成完整回路。

负极（锌）：Zn → Zn²⁺ + 2e⁻ (氧化反应)
正极（铜）：2H⁺ + 2e⁻ → H₂↑ (还原反应)
总反应：Zn + 2H⁺ → Zn²⁺ + H₂↑
阴极（铜）：Cu²⁺ + 2e⁻ → Cu (还原反应)
阳极（锌）：Zn → Zn²⁺ + 2e⁻ (氧化反应)
总反应：Zn + Cu²⁺ → Zn²⁺ + Cu
